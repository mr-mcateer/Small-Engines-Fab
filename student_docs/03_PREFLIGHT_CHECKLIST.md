# DRIFT TRIKE PRE-RIDE SAFETY CHECKLIST
## MUST BE COMPLETED BEFORE EVERY RIDE
### Colton & Atticus - No Exceptions

---

## RIDER EQUIPMENT
- [ ] DOT helmet (full face recommended)
- [ ] Gloves
- [ ] Long pants (no shorts, no loose clothing)
- [ ] Closed-toe shoes or boots
- [ ] No dangling jewelry, hoodie strings, etc.

## ENGINE
- [ ] Oil level checked (dipstick between marks)
- [ ] No visible oil leaks
- [ ] No visible fuel leaks
- [ ] Air filter secure and clean
- [ ] Exhaust tight, no leaks at header
- [ ] Spark plug wire secure
- [ ] All engine mounting bolts tight

## SAFETY SYSTEMS
- [ ] Kill switch (handlebar) - TEST IT: start engine, press kill, engine must die
- [ ] Tether kill switch - TEST IT: start engine, pull tether, engine must die
- [ ] Throttle return - TEST IT: twist throttle open, release - must snap to closed
- [ ] Throttle cable - no fraying, smooth operation, no binding

## BRAKES
- [ ] Front brake lever - squeeze and hold, wheel must not spin
- [ ] Rear brake lever/pedal - activate and hold, wheels must not spin
- [ ] Brake cables/lines - no fraying, no kinks, properly routed

## DRIVETRAIN
- [ ] Chain tension correct (1/2" to 3/4" deflection at mid-span)
- [ ] Chain guard in place
- [ ] All sprocket bolts tight
- [ ] Torque converter / clutch bolts tight
- [ ] Axle lock collars tight

## FRAME AND WHEELS
- [ ] All visible welds - no cracks
- [ ] Steering moves freely, no binding
- [ ] Headset tight (no play in handlebars when braking)
- [ ] Front wheel axle tight
- [ ] Rear wheel hubs tight on axle
- [ ] Tire pressure (if pneumatic) - check with gauge
- [ ] PVC drift sleeves secure (if installed)
- [ ] Seat bolts tight

## ENVIRONMENT
- [ ] Mr. McAteer present and aware
- [ ] Fire extinguisher within 50 feet
- [ ] Riding area clear of pedestrians
- [ ] Riding area: flat, paved, no traffic
- [ ] No fuel on clothing or ground near trike
